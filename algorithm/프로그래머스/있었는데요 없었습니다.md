# 코드

```sql
SELECT A1.ANIMAL_ID , A1.NAME FROM ANIMAL_OUTS A1 
JOIN ANIMAL_INS A2 
ON A1.ANIMAL_ID = A2.ANIMAL_ID AND A2.DATETIME > A1.DATETIME
ORDER BY A2.DATETIME ASC

```
