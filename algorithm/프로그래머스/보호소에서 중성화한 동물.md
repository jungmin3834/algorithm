# 코드

```sql
SELECT ANIMAL_ID , ANIMAL_TYPE , NAME FROM ANIMAL_INS A
WHERE 0 < (SELECT COUNT(*) FROM ANIMAL_OUTS O WHERE O.SEX_UPON_OUTCOME!=A.SEX_UPON_INTAKE AND O.ANIMAL_ID = A.ANIMAL_ID)
```
