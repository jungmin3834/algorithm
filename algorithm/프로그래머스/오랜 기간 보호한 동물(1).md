# 코드

```cpp
SELECT DISTINCT A.NAME , A.DATETIME FROM ANIMAL_INS A 
WHERE 0=(SELECT COUNT(*) FROM ANIMAL_OUTS O WHERE A.ANIMAL_ID=O.ANIMAL_ID)
GROUP BY A.DATETIME ASC LIMIT 3
```
