# 코드

```cpp
class Solution {
public:
    double angleClock(int hour, int minutes) {
        double hAngle = hour % 12 * 30 + (minutes * 0.5);
        double mAngle = minutes * 6;
        return min(abs(mAngle - hAngle),360 - abs(mAngle - hAngle));
    }
};

```
