# 코드

```cpp
class Solution {
public:
    bool isPerfectSquare(int num) {
        for(long i = 0;i<=num;i++){
            if(i * i == num)
                return true;
            else if(i * i > num)
                return false;
        }
        return false;
     }
};
```
